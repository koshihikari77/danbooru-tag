# Danbooru Tag Hierarchy Scraper - プロジェクト概要

## 目的
画像に付与されているタグから人物の可変的特徴（目の色、髪の色、胸の大きさなど）を特定・分離し、別の人物特徴への置換を支援する。

## 主な機能
1. DanbooruのTag Groupsと個別タグの階層構造を抽出
2. 包括的な親子関係データベースを構築
3. 人物固有特徴の自動分類システム
4. LlamaIndexを使った自然言語によるタグ検索

## 技術スタック
- **言語**: Python 3.12+
- **依存関係管理**: uv
- **データ処理**: polars, pandas
- **HTMLパーサー**: BeautifulSoup4
- **HTTP取得**: curl (subprocess経由)
- **AI検索**: LlamaIndex
- **ログ**: 標準logging

## 現在の成果
- 15,183件のタグ階層データを抽出・分類完了
- 人物固有特徴2,265件、非人物固有特徴4,991件を自動分類
- 上位20,000件の画像データを時間当たりスコアで抽出
